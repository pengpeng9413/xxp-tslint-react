# xxp-tslint-react
一个让自己写代码舒服的tslint规范

## TSLint Config for react developer

## Installation

```sh
yarn add tslint-config-xxp --dev
```

## Usage

In `tslint.json`:

```json
{
  "extends": "tslint-config-xxp"
}
```

### Rules

- [TSLint](https://www.npmjs.com/package/tslint)
- [TSLint React Rules](https://github.com/palantir/tslint-react)
